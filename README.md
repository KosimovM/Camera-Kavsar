# Camera Kavsar
Cavsar Kamera — веб-приложение барои камера/сабти видео бо интерфейси оддӣ ва муосир.
🚀 Features
✅ Access camera device
✅ Capture photo
✅ Record video
✅ Modern UI design
✅ Responsive layout
✅ Easy to use
🛠 Technologies
Next.js
TypeScript
Tailwind CSS
WebRTC API
📦 Installation
git clone project-url
cd project-folder
npm install
npm run dev
🌐 Usage
Open project in browser
Allow camera permission
Start capturing photos or video
⚠️ Requirements
Modern browser (Chrome, Edge, Safari latest)
Camera permission enabled
